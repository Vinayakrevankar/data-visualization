# Oscars Dashboard PRO++

React + D3 dashboard with multiple titled charts and hover tooltips. Your real Oscars dataset is embedded at `src/data/oscars.csv`.

## Run
```bash
npm install
npm run dev
```

## Charts
- Category Distribution by Decade (stacked, counts ↔ percentages, tooltips, sliders)
- Award Trends Over Years (line with hover)
- Category Popularity Heatmap (decade × category)
- Category–Film Treemap
- Class–Category–Film Sunburst
- Top Award Winners (slider, hover)
- Category Share Pie
- Films Per Decade Bar
